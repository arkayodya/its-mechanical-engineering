# 05. Major Projects

This directory contains the most significant engineering projects completed during my undergraduate studies.

Projects included here represent multidisciplinary work that integrates design, analysis, manufacturing, programming, experimentation, and documentation.

## Project Standard

Each project should include:

- README
- Objectives
- CAD Files
- Engineering Drawings
- Source Code (if applicable)
- Simulation Files
- Photographs
- Technical Report
- Lessons Learned
- Future Improvements

Only projects that meet professional documentation standards are included.
