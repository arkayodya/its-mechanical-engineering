# Project: Boeing 737-800 Aircraft Model ✈️

## 📌 Project Overview

This project documents my first long-term Autodesk Fusion 360 project, following a guided tutorial while learning professional CAD workflows.

Rather than simply recreating an aircraft, the objective is to develop a strong foundation in parametric CAD, reference-based modeling, surface generation, and engineering documentation.

The aircraft chosen is the **Boeing 737-800**, modeled using accurately calibrated three-view blueprints and real-world dimensions.

- **Part Number:** `ITS-ME-01-002`
- **Software Used:** Autodesk Fusion 360
- **Project Type:** CAD Learning Project
- **Status:** 🚧 In Progress

---

## 🎯 Learning Objectives

This project aims to develop competency in:

- Parametric CAD workflows
- Blueprint calibration and reference management
- Construction geometry
- Sketch constraints
- T-Spline and surface modeling
- Loft and form generation
- Assembly organization
- Engineering documentation

## 🛠️ Engineering Workflow

The project follows a structured engineering workflow rather than directly creating geometry.

1. Prepare high-quality reference material.
2. Calibrate blueprints using real-world aircraft dimensions.
3. Organize construction geometry.
4. Build major aircraft components.
5. Refine geometry.
6. Export manufacturing-neutral CAD formats.
7. Document the complete design process.

## 📝 Lessons Learned

### Session 01 — Blueprint Preparation

Completed the entire project setup by preparing accurately scaled reference canvases.

Key takeaways:

- Orthographic projection is essential for accurate blueprint alignment.
- Proper blueprint calibration eliminates scaling issues later in the design process.
- Construction planes help maintain an organized modeling workspace.
- Spending additional time preparing references significantly improves downstream CAD workflow.
