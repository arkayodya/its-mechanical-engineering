# Parametric Spur Gear Assembly ⚙️

## 📌 Overview

This project marks my first experience with computer-aided design (CAD) and serves as the starting point of my mechanical engineering portfolio.

Completed during the pre-university self-study period, the project explores the fundamentals of parametric modeling through the design and assembly of a pair of spur gears mounted on a common base plate. The emphasis was placed on understanding gear geometry, assembly constraints, and basic mechanical design principles rather than producing a manufacturing-ready component.

## 🎯 Objectives

- Learn the fundamentals of parametric CAD modeling.
- Design a mating pair of involute spur gears.
- Create a simple mechanical assembly with proper center-distance alignment.
- Understand rotational relationships between meshing gears.
- Establish a personal CAD workflow for future engineering projects.

This project introduced several core mechanical design concepts, including:

- Assembly constraints
- Center-distance alignment
- Basic interference checking
- STEP (`.step`) export for CAD interoperability
