# 07. Templates

Consistency is a hallmark of professional engineering documentation.

This directory contains reusable templates for future projects.

Examples include:

- Project README
- Technical Report
- Design Review
- Meeting Notes
- Lessons Learned
- Decision Log

Using templates ensures every project follows the same documentation standard.
