# 04. Engineering Notes 📚

This directory serves as my personal mechanical engineering knowledge base throughout university.

Rather than storing lecture slides or copied class notes, it contains my own explanations, derivations, diagrams, summaries, and reflections. Every note is written with the goal of understanding concepts deeply enough to apply them in future engineering projects.

Over time, this collection will grow into a permanent reference that connects theory with practical engineering work documented elsewhere in this repository.

---

## 🎯 Objectives

- Build a strong foundation in core engineering principles.
- Reinforce learning through active explanation and problem solving.
- Connect theoretical concepts to engineering projects and real-world applications.
- Develop a searchable knowledge base that continues to grow beyond university.

---

---

## 📖 Note Philosophy

Every note should answer one question:

> **Could future me understand this concept using only this page?**

If the answer is yes, then the note has achieved its purpose.

Whenever possible, notes should include:

- Clear explanations in my own words
- Diagrams or illustrations
- Worked examples
- Links to related concepts
- Connections to engineering projects within this repository
